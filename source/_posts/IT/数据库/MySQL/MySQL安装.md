# MySQL安装

一、安装[mysql80.rpm](https://dev.mysql.com/downloads/repo/yum/)

{%spoiler 示例代码%}
```bash
rpm -i mysql80-community-release-el8-1.noarch.rpm
```
{%endspoiler%}

二、yum安装

{%spoiler 示例代码%}
```bash
yum install mysql-server
```
{%endspoiler%}
